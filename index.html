Welcome to Kartmakare

Kartmakare needs to store data on your computer to work. The data will only be accessed by the tool and nothing you do in the tool will be shared with Twofour.

We will also count how many times this tool is used and when it is used, but nothing else. This is so that we can see if it's popular, and if we should maybe create other tools in the future.

Is this okay?

I changed my mind, Kartmakare may store data on my computer, and it's okay that you see if it's popular by logging when this URL is opened.
